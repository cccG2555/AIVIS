# 🏭 산업안전 AI 비전 모니터링 시스템 (Industrial Safety AI Vision)

![Version](https://img.shields.io/badge/version-v2.0-blue)
![Python](https://img.shields.io/badge/python-3.8%2B-blue)
![YOLO](https://img.shields.io/badge/YOLO-v8-yellow)

본 프로젝트는 건설 현장, 제조 공장 등 산업 현장에서 발생할 수 있는 중대재해를 예방하기 위해 개발된 **지능형 AI 비전 모니터링 시스템**입니다. CCTV 및 웹캠 영상을 실시간으로 분석하여 작업자의 안전 상태를 추적하고 위급 상황 발생 시 즉각적인 경고를 발생시킵니다.

## ✨ 주요 기능 (Key Features)

### 1. 🦺 안전 장비(PPE) 착용 실시간 모니터링
- 작업자의 헬멧(Helmet) 및 안전조끼(Vest) 착용 여부를 실시간으로 감지합니다.
- **소유권 매칭 알고리즘**: 바닥에 방치된 장비와 실제 착용한 장비를 구분하여(Margin Matching), 실제 미착용 부위에만 정확히 붉은색 경고 박스를 동적으로 표시합니다.

### 2. 🚨 작업자 쓰러짐(Fall) 감지 시스템
- 척추 중심선의 각도와 사람 바운딩 박스의 가로세로 비율(Aspect Ratio)을 교차 검증합니다.
- 단순한 허리 숙임(신발끈 묶기 등)과 실제 쓰러짐을 완벽히 구분합니다.
- **20초 유예 타이머**: 쓰러진 자세가 감지되면 즉시 알람을 울리는 대신 화면에 초시계를 표시하며, 움직임 없이 20초가 유지될 때만 최종 알람을 발동하여 불필요한 알람 공해를 방지합니다.

### 3. 🆘 긴급 구조(SOS) 동작 감지
- 작업자가 위급 상황 시 양팔을 교차하여 머리 위로 X자를 만드는 동작을 인식합니다.
- **가상 손끝 연장선 및 데드존**: 뼈대(Keypoint)의 팔뚝 벡터를 80% 연장하여 실제 손끝의 교차를 계산하며, 우연한 겹침(태블릿 사용 등)을 철저히 차단합니다.
- 5초 이내 3회 동작 달성 시 알람이 발동되며, 0.3초 쿨다운을 적용해 미세 떨림으로 인한 오작동 카운트를 방지합니다.

## 🛠 핵심 최적화 기술 (Engineering Highlights)

- **비동기식 캐싱 렌더링 (Asynchronous Rendering Cache)**: 무거운 AI 추론(Inference)은 N프레임마다 백그라운드에서 실행하고, 화면 렌더링은 캐시된 데이터를 사용해 매 프레임 그려냅니다. 연산 부하를 50% 절감하면서도 30~60FPS의 부드러운 화면을 유지합니다.
- **NMS (Non-Maximum Suppression) 겹침 필터링**: 작업자가 누웠을 때 AI가 다리와 상체를 분리해 2명으로 중복 인식하는 환각 현상을 IoU(겹침 면적) 계산으로 융합(Merge)시킵니다.
- **화면 밖 이탈 오작동 차단**: 관절 좌표가 화면 테두리를 벗어날 때 발생하는 지그재그 뼈대 렌더링과 가짜 알람을 확신도(Confidence) 및 화면 경계선(Boundary) 필터링으로 완벽히 차단합니다.

## ⚙️ 시스템 파라미터 튜닝 (Configuration)

코드 상단의 `[시스템 파라미터 설정 구역]`에서 현장 상황에 맞춰 민감도를 즉시 조절할 수 있습니다. 비개발자도 손쉽게 세팅이 가능합니다.

```python
# [AI 모델 신뢰도]
CONF_HELMET_VEST = 0.60  # 장비 인식 커트라인
CONF_PERSON      = 0.60  # 사람 인식 커트라인
FRAME_INTERVAL   = 5     # 5프레임당 1번 AI 연산 수행

# [쓰러짐 감지 파라미터]
FALL_ANGLE_LIMIT     = 45.0  # 척추 각도 제한
FALL_ASPECT_RATIO    = 0.8   # 가로 폭 비율 제한 (구부림 방지용)
FALL_MOTIONLESS_TIME = 20.0  # 기절 확정 대기 타이머 (초)

# [SOS 감지 파라미터]
SOS_LIMIT_TIME = 5.0     # 3회 달성 제한 시간 (초)
SOS_COOLDOWN   = 0.3     # 떨림 방지 대기 시간 (초)
```

## 🚀 시작하기 (Getting Started)

### 요구 사항 (Prerequisites)
- Python 3.8 이상 권장
- `ultralytics` (YOLO 라이브러리)
- `opencv-python`

### 설치 및 실행
1. 필요한 패키지를 설치합니다.
   ```bash
   pip install ultralytics opencv-python
   ```
2. 학습된 커스텀 장비 모델 파일(`best.pt`)과 자세 추정 모델(`yolo26m-pose.pt` 또는 `yolov8n-pose.pt`)을 파이썬 스크립트와 동일한 폴더에 위치시킵니다.
3. 메인 스크립트를 실행합니다.
   ```bash
   python safety_factory_v1.py
   ```
4. 실행 후 `q` 키를 누르면 시스템이 안전하게 종료됩니다.
